Welcome to Twootstrap.

Twootstrap is a simple Kohana Asset manager for Kohana framework you can easily manage all your assets with this asset manager. The easiest may to install this is through command line.

Getting The Twootstrap Module

Using comman line you can do the following...

# Change to your kohana modules directory
$ cd kohana/modules
$ git clone https://github.com/CreativityKills/Twootstrap.git

You can also click on the Zip and TarBall links at the top of the page to download the pack, then unzip it to your modules folder.

Activating Your Module

After you have put Twootstrap into your modules folder, then edit your application/bootstrap.php file.

Kohana::modules(array(
    // ...other modules...
    'twootstrap' => MODPATH.'twootstrap', // Twootstrap asset manager module
    // ..other modules...
));
